carsten-receiver
================

Carsten Receiver allows you to fetch from carsten and display stuff.

run carsten-receiver
--------------------

```
git clone https://github.com/carst-it/carsten-receiver.git
cd carsten-receiver
npm install -g gulp
npm install
npm start
```

build carsten-receiver
----------------------

```
gulp
```
